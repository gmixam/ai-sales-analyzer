Internal Operator UI

Manual Report Run

Минимальный операторский интерфейс для ручного запуска manager_daily и rop_weekly поверх текущего bounded reporting API path.

Параметры запуска

Список менеджеров берётся из локального Bitrix-synced справочника. Для актуализации списка нажмите Refresh Local Managers. Telegram test delivery для manual runs выполняется всегда в operator chat. Checkbox управляет только business email delivery для менеджера / РОП. Resolved email recipients показываются как reference.

Resolved Recipients

Нажмите Preview Recipients, чтобы увидеть resolved targets.

Run Result

После запуска здесь появятся общий статус, report statuses и blocking reasons.

Run Observability

Операторский мониторинг текущего manual run: состояние, этапы, итог и cost только по реально использованным AI-узлам.

Run-state появится после запуска.

Stages

После запуска здесь появятся source-discovery, persistence-check, ingest-missing, audio-fetch, STT, analysis, report-build/render и delivery.

Run Summary

Итоговая summary появится после запуска.

AI Cost

Если в run использовались AI-узлы, здесь появится cost block. Если metadata нет, будет показан safe fallback.

Diagnostics

После запуска здесь появятся effective filters, reason codes и selection transparency.
